# VSGruvbox
A UI theme for Visual Studio that uses the [gruvbox](https://github.com/morhetz/gruvbox) color scheme. Currently only the dark theme is supported.

## Installation

First import `Gruvbox.vstheme` with [the Visual Studio theme plugin](https://visualstudiogallery.msdn.microsoft.com/6f4b51b6-5c6b-4a81-9cb5-f2daa560430b).

Then install the text editor colorscheme by importing gruvbox.vstheme
